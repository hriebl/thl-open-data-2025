### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run einwohner.py
   ```

   or

   ```
   $ streamlit run wasserstand_vorlesung.py
   ```

   or

   ```
   $ streamlit run wasserstand_verbessert.py
   ```
